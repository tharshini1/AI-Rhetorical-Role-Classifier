# AI-Rhetorical-Role-Classifier
# ⚖️ AI-Powered Rhetorical Role Classification

An AI-powered web application that classifies legal text into rhetorical roles using a fine-tuned Transformer model. Built with **Flask**, **Hugging Face Transformers**, and **PyTorch**, the application provides an intuitive interface for analyzing legal documents.

---

## 🚀 Features

- 🧠 AI-based rhetorical role prediction
- 📄 Analyze legal text instantly
- ⚡ Fast inference using Hugging Face Transformers
- 🌐 Interactive Flask web interface
- 📊 Displays prediction confidence
- 📱 Responsive and user-friendly design
- 🔒 Robust error handling
- ☁️ Ready for GitHub and Render deployment

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Rhetorical-Role-Classifier.git
```

Navigate to the project folder

```bash
cd Rhetorical-Role-Classifier
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
